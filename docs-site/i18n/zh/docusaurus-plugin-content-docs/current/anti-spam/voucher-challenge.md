# 优惠券挑战

优惠券挑战是一种反垃圾邮件机制，可通过独特的优惠券代码来控制内容发布。它将信任转移给社区所有者，社区所有者手动将代码分发给他们信任的人。

**源代码：** [github.com/bitsocialnet/voucher-challenge](https://github.com/bitsocialnet/voucher-challenge)

## 如何运作

1. 社区所有者生成一个或多个唯一的优惠券代码。
2. 所有者通过他们选择的渠道（直接消息、电子邮件、亲自等）。
3. 当作者尝试发布时，挑战系统会提示他们输入优惠券代码。
4. 代码经过验证 - 如果它是真实的且尚未使用，则出版物被接受。

每个优惠券代码在兑换后都与特定作者绑定，防止其他人重复使用。

## 何时使用

优惠券挑战最适合：

- **仅限邀请的社区**，其中会员资格受到有意限制。
- **策划空间**，所有者亲自审核每个参与者。
- **高信任环境**，其中自动垃圾邮件评分是不必要或不可取的。

因为它需要手动代码分发，所以它无法扩展到大型开放社区。对于这些场景，请考虑使用 [垃圾邮件拦截器](./spam-blocker.md) 或 [EVM 合约调用挑战](./evm-contract-call.md)。

## 集成

Voucher Challenge 插入与 Bitsocial 生态系统中其他反垃圾邮件包使用的相同挑战接口。社区所有者通过其社区设置启用它，当作者尝试发帖时，会自动向他们提出挑战。
