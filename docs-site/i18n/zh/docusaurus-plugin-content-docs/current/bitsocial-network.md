# Bitsocial网络

Bitsocial Network 是总体规划中用于位于 Bitsocial 应用程序下的共享应用链层的名称。

这个想法并不是要取代点对点社交层。它旨在提供社交应用程序在大规模与现有平台竞争时所需的共享经济和协调原语。

## 这意味着什么

- 去中心化的 Bitsocial 领域，例如 `.bso`
- 跨应用程序共享流动性
- 持久的货币化轨道
- 银行或平台更难以遏制的金融结构
- 不依赖于一家公司拥有整个堆栈的网络效应

## 为什么重要

分散社区和身份是必要的，但分散所有社交媒体还不够。

如果经济层仍然依赖于少数集中化轨道，那么应用程序仍然很容易受到压力、平台化或财务匮乏。 Bitsocial Network 是针对第二层依赖关系的建议答案。

## 与 Bitsocial 其余部分的关系

Bitsocial Network 被提议作为 Bitsocial 应用程序的共享基础设施，而不是作为用户每天直接交互的产品。

面向公众的结果将是：

- 社区保持点对点关系
- 应用程序保持差异化
- 货币化和命名变得更加持久
- 价值可以在整个生态系统中移动，而无需重新创建中心化平台所有者

## 为什么这位于第4阶段

总体规划将其置于早期应用程序级证明点之后：

1. 去中心化图像板
2. 启动Bitsocial RPC
3. 去中心化论坛
4. 启动Bitsocial网络

这个顺序是有意义的，因为一旦网络上已经存在真正的应用程序和真实的用户行为，生态系统级别的协调层就会更有用。
