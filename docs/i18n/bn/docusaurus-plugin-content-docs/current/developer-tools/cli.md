# সিএলআই

`bitsocial-cli` হল Bitsocial প্রোটোকল ব্যাকএন্ডের সাথে ইন্টারঅ্যাক্ট করার জন্য একটি কমান্ড-লাইন টুল। এটি আপনাকে একটি স্থানীয় P2P ডেমন চালাতে, সম্প্রদায়গুলি তৈরি এবং কনফিগার করতে এবং সামগ্রী প্রকাশ করতে দেয় -- সব টার্মিনাল থেকে।

এটি ভাগ করা বিটসোশ্যাল প্রোটোকল ক্লায়েন্ট লেয়ারের উপরে তৈরি করা হয়েছে এবং সম্প্রদায় তৈরি এবং নোড পরিচালনার জন্য [5chan](/apps/5chan/) এবং [সিডিট](/apps/seedit/) দ্বারা ব্যবহৃত হয়।

## ইনস্টলেশন

পূর্ব-নির্মিত বাইনারিগুলি Windows, macOS এবং Linux-এর জন্য উপলব্ধ। GitHub থেকে আপনার প্ল্যাটফর্মের জন্য সর্বশেষ রিলিজ ডাউনলোড করুন:

**[GitHub রিলিজ থেকে ডাউনলোড করুন](https://github.com/bitsocialnet/bitsocial-cli/releases)**

ডাউনলোড করার পরে, বাইনারি এক্সিকিউটেবল করুন (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## ডেমন চালাচ্ছে

CLI-এর সবচেয়ে সাধারণ ব্যবহার হল বিটসোশ্যাল নোড চালানো। ডেমন P2P নেটওয়ার্কিং স্তর শুরু করে এবং একটি স্থানীয় API প্রকাশ করে যা ক্লায়েন্টরা সংযোগ করতে পারে।

```bash
bitsocial-cli daemon
```

প্রথম লঞ্চে, ডেমন আউটপুট **WebUI**-এর সাথে লিঙ্ক করে, আপনার নোড, সম্প্রদায় এবং সেটিংস পরিচালনার জন্য একটি ব্রাউজার-ভিত্তিক গ্রাফিকাল ইন্টারফেস। আপনি যদি টার্মিনাল কমান্ডের চেয়ে একটি GUI পছন্দ করেন তবে এটি কার্যকর।

## কী অ্যাকশন

| কর্ম                              | বর্ণনা                                               |
| --------------------------------- | ---------------------------------------------------- |
| ডেমন শুরু করুন                    | Bitsocial P2P নোড চালু করুন                          |
| একটি সম্প্রদায় তৈরি করুন         | একটি নতুন সম্প্রদায় তৈরি করুন                       |
| একটি সম্প্রদায় সম্পাদনা করুন     | সম্প্রদায় সেটিংস আপডেট করুন (শিরোনাম, বিবরণ, নিয়ম) |
| স্থানীয় সম্প্রদায়ের তালিকা করুন | এই নোডে হোস্ট করা সম্প্রদায়ের তালিকা করুন           |
| একটি সম্প্রদায় শুরু করুন         | একটি নির্দিষ্ট সম্প্রদায়ের সেবা করা শুরু করুন       |
| একটি সম্প্রদায় বন্ধ করুন         | একটি নির্দিষ্ট সম্প্রদায়ের সেবা করা বন্ধ করুন       |

আপনার ইনস্টল করা রিলিজ দ্বারা প্রকাশিত বর্তমান কমান্ডের নাম এবং পতাকাগুলি দেখতে `--help` এর সাথে CLI চালান:

```bash
bitsocial-cli --help
bitsocial-cli daemon --help
```

## সাধারণ কর্মপ্রবাহ

একটি নতুন সম্প্রদায় হোস্ট করার জন্য একটি সাধারণ সেটআপ প্রবাহ:

```bash
# 1. Start the daemon
bitsocial-cli daemon

# 2. In another terminal, inspect the available community-management commands
bitsocial-cli --help
```

সেখান থেকে, একটি সম্প্রদায় তৈরি করতে, কনফিগার করতে এবং পরিষেবা দেওয়া শুরু করতে ইনস্টল করা রিলিজের সম্প্রদায়-ব্যবস্থাপনা কমান্ডগুলি ব্যবহার করুন৷ একবার শুরু হলে, সম্প্রদায়টি বিটসোশ্যাল নেটওয়ার্কে লাইভ এবং সামঞ্জস্যপূর্ণ ক্লায়েন্টদের কাছ থেকে অ্যাক্সেসযোগ্য।
