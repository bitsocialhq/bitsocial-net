# CLI

Το `bitsocial-cli` είναι ένα εργαλείο γραμμής εντολών για αλληλεπίδραση με το backend του πρωτοκόλλου Bitsocial. Σας επιτρέπει να εκτελείτε έναν τοπικό δαίμονα P2P, να δημιουργείτε και να διαμορφώνετε κοινότητες και να δημοσιεύετε περιεχόμενο -- όλα από το τερματικό.

Είναι χτισμένο πάνω από το κοινό επίπεδο πελάτη πρωτοκόλλου Bitsocial και χρησιμοποιείται από τα [5καν](/apps/5chan/) και [Seedit](/apps/seedit/) για τη δημιουργία κοινότητας και τη διαχείριση κόμβων.

## Εγκατάσταση

Τα προκατασκευασμένα δυαδικά αρχεία είναι διαθέσιμα για Windows, macOS και Linux. Κατεβάστε την πιο πρόσφατη έκδοση για την πλατφόρμα σας από το GitHub:

**[Λήψη από τις εκδόσεις GitHub](https://github.com/bitsocialnet/bitsocial-cli/releases)**

Μετά τη λήψη, κάντε το δυαδικό εκτελέσιμο (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## Τρέχοντας τον Δαίμονα

Η πιο κοινή χρήση του CLI είναι η εκτέλεση ενός κόμβου Bitsocial. Ο δαίμονας ξεκινά το επίπεδο δικτύωσης P2P και εκθέτει ένα τοπικό API στο οποίο μπορούν να συνδεθούν οι πελάτες.

```bash
bitsocial-cli daemon
```

Κατά την πρώτη εκκίνηση, ο δαίμονας εξάγει συνδέσμους προς το **WebUI**, μια γραφική διεπαφή που βασίζεται σε πρόγραμμα περιήγησης για τη διαχείριση του κόμβου, των κοινοτήτων και των ρυθμίσεών σας. Αυτό είναι χρήσιμο εάν προτιμάτε ένα GUI από τις εντολές τερματικού.

## Βασικές Ενέργειες

| Δράση                        | Περιγραφή                                                       |
| ---------------------------- | --------------------------------------------------------------- |
| Ξεκινήστε τον δαίμονα        | Εκκινήστε τον κόμβο Bitsocial P2P                               |
| Δημιουργία κοινότητας        | Δημιουργία νέας κοινότητας                                      |
| Επεξεργασία κοινότητας       | Ενημέρωση ρυθμίσεων κοινότητας (τίτλος, περιγραφή, κανόνες)     |
| Κατάλογος τοπικών κοινοτήτων | Καταχωρίστε τις κοινότητες που φιλοξενούνται σε αυτόν τον κόμβο |
| Ξεκινήστε μια κοινότητα      | Ξεκινήστε να εξυπηρετείτε μια συγκεκριμένη κοινότητα            |
| Σταματήστε μια κοινότητα     | Σταματήστε να εξυπηρετείτε μια συγκεκριμένη κοινότητα           |

Εκτελέστε το CLI με το `--help` για να δείτε τα τρέχοντα ονόματα εντολών και σημαίες που εκτίθενται από την εγκατεστημένη έκδοση:

```bash
bitsocial-cli --help
bitsocial-cli daemon --help
```

## Τυπική ροή εργασίας

Μια κοινή ροή εγκατάστασης για τη φιλοξενία μιας νέας κοινότητας:

```bash
# 1. Start the daemon
bitsocial-cli daemon

# 2. In another terminal, inspect the available community-management commands
bitsocial-cli --help
```

Από εκεί, χρησιμοποιήστε τις εντολές διαχείρισης κοινότητας της εγκατεστημένης έκδοσης για να δημιουργήσετε, να διαμορφώσετε και να ξεκινήσετε την προβολή μιας κοινότητας. Μόλις ξεκινήσει, η κοινότητα είναι ζωντανή στο δίκτυο Bitsocial και είναι προσβάσιμη από συμβατούς πελάτες.
