# CLI

`bitsocial-cli` یک ابزار خط فرمان برای تعامل با باطن پروتکل Bitsocial است. این به شما امکان می دهد یک شبح P2P محلی را اجرا کنید، انجمن ها را ایجاد و پیکربندی کنید، و محتوا را منتشر کنید - همه از ترمینال.

این در بالای لایه مشتری پروتکل Bitsocial مشترک ساخته شده است و توسط [5chan](/apps/5chan/) و [بینید](/apps/seedit/) برای ایجاد انجمن و مدیریت گره استفاده می شود.

## نصب و راه اندازی

باینری های از پیش ساخته شده برای ویندوز، macOS و لینوکس در دسترس هستند. آخرین نسخه را برای پلتفرم خود از GitHub دانلود کنید:

**[دانلود از GitHub Releases](https://github.com/bitsocialnet/bitsocial-cli/releases)**

پس از دانلود، باینری را قابل اجرا (macOS/Linux) کنید:

```bash
chmod +x bitsocial-cli
```

## اجرای دیمون

رایج ترین استفاده از CLI اجرای یک گره Bitsocial است. دیمون لایه شبکه P2P را راه اندازی می کند و یک API محلی را نشان می دهد که مشتریان می توانند به آن متصل شوند.

```bash
bitsocial-cli daemon
```

در اولین راه‌اندازی، دیمون پیوندهایی را به **WebUI**، یک رابط گرافیکی مبتنی بر مرورگر برای مدیریت گره، جوامع و تنظیمات شما خروجی می‌دهد. اگر یک رابط کاربری گرافیکی را به دستورات ترمینال ترجیح می دهید، مفید است.

## اقدامات کلیدی

| اقدام                  | توضیحات                                             |
| ---------------------- | --------------------------------------------------- | ------------------ |
| دیمون را شروع کنید     | گره Bitsocial P2P                                   | را راه اندازی کنید |
| ایجاد یک انجمن         | ایجاد یک انجمن جدید                                 |
| ویرایش یک انجمن        | به روز رسانی تنظیمات انجمن (عنوان، توضیحات، قوانین) |
| فهرست جوامع محلی       | فهرست انجمن های میزبانی شده در این گره              |
| راه اندازی یک انجمن    | شروع خدمت به یک جامعه خاص                           |
| یک جامعه را متوقف کنید | توقف خدمت به یک جامعه خاص                           |

CLI را با `--help` اجرا کنید تا نام‌ها و پرچم‌های دستور فعلی را که توسط نسخه نصب شده شما در معرض دید قرار گرفته‌اند، مشاهده کنید:

```bash
bitsocial-cli --help
bitsocial-cli daemon --help
```

## گردش کار معمولی

یک جریان راه اندازی رایج برای میزبانی یک جامعه جدید:

```bash
# 1. Start the daemon
bitsocial-cli daemon

# 2. In another terminal, inspect the available community-management commands
bitsocial-cli --help
```

از آنجا، از دستورات مدیریت انجمن نسخه نصب شده برای ایجاد، پیکربندی و شروع به ارائه یک انجمن استفاده کنید. پس از شروع، انجمن در شبکه Bitsocial فعال است و از مشتریان سازگار قابل دسترسی است.
