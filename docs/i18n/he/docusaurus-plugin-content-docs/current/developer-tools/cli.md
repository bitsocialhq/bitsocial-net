# CLI

ה-`bitsocial-cli` הוא כלי שורת פקודה לאינטראקציה עם הקצה האחורי של פרוטוקול Bitsocial. זה מאפשר לך להפעיל דמון P2P מקומי, ליצור ולהגדיר קהילות ולפרסם תוכן - הכל מהמסוף.

הוא בנוי על גבי שכבת הלקוח המשותפת של פרוטוקול Bitsocial ומשמש את [5chan](/apps/5chan/) ו-[Seedit](/apps/seedit/) ליצירת קהילה וניהול צמתים.

## התקנה

קבצים בינאריים מובנים מראש זמינים עבור Windows, macOS ו-Linux. הורד את המהדורה האחרונה עבור הפלטפורמה שלך מ-GitHub:

**[הורד מ-GitHub מהדורות](https://github.com/bitsocialnet/bitsocial-cli/releases)**

לאחר ההורדה, הפוך את קובץ ההפעלה הבינארי (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## מפעיל את הדימון

השימוש הנפוץ ביותר ב-CLI הוא הפעלת צומת Bitsocial. הדמון מתחיל את שכבת הרשת P2P וחושף API מקומי שלקוחות יכולים להתחבר אליו.

```bash
bitsocial-cli daemon
```

בהפעלה הראשונה, הדמון מוציא קישורים ל-**WebUI**, ממשק גרפי מבוסס דפדפן לניהול הצומת, הקהילות וההגדרות שלך. זה שימושי אם אתה מעדיף GUI על פני פקודות מסוף.

## פעולות מפתח

| פעולה                | תיאור                                     |
| -------------------- | ----------------------------------------- |
| התחל את הדמון        | הפעל את הצומת Bitsocial P2P               |
| צור קהילה            | צור קהילה חדשה                            |
| ערוך קהילה           | עדכון הגדרות הקהילה (כותרת, תיאור, כללים) |
| רשימת קהילות מקומיות | רשימת קהילות המתארחות בצומת זה            |
| התחל קהילה           | התחל לשרת קהילה ספציפית                   |
| עצור קהילה           | הפסק לשרת קהילה ספציפית                   |

הפעל את ה-CLI עם `--help` כדי לראות את שמות הפקודות והדגלים הנוכחיים שנחשפו על ידי המהדורה המותקנת שלך:

```bash
bitsocial-cli --help
bitsocial-cli daemon --help
```

## זרימת עבודה טיפוסית

זרימת הגדרה נפוצה לאירוח קהילה חדשה:

```bash
# 1. Start the daemon
bitsocial-cli daemon

# 2. In another terminal, inspect the available community-management commands
bitsocial-cli --help
```

משם, השתמש בפקודות ניהול הקהילה של המהדורה המותקנת כדי ליצור, להגדיר ולהתחיל לשרת קהילה. לאחר תחילתה, הקהילה פעילה ברשת Bitsocial ונגישה מלקוחות תואמים.
